# Rivet Node Reference

The node reference contains information about the nodes built into Rivet.

For an overview of all nodes, see the [All Nodes Page](./node-reference/all-nodes.mdx).

For new chat graphs, start with [LLM Chat](./node-reference/llm-chat.mdx). The older [Chat](./node-reference/chat.mdx) node remains available for existing OpenAI-compatible graphs.

If you are looking for a plugin node, install the plugin in Rivet Settings > Plugins and refer to that plugin's documentation, or check the [built-in plugins documentation](./user-guide/plugins/built-in/all.md).

Otherwise, find your node in the sidebar to the left.
