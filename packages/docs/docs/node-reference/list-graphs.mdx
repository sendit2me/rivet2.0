# List Graphs Node

The List Graphs node returns references to every graph in the current project.

Use it when a workflow needs to inspect or dynamically call project graphs.

## Inputs

This node has no inputs.

## Outputs

| Title       | Data Type           | Description                                     |
| ----------- | ------------------- | ----------------------------------------------- |
| Graphs      | `graph-reference[]` | References for all graphs in the project.       |
| Graph Names | `string[]`          | The display names of all graphs in the project. |

## Settings

This node has no settings.
