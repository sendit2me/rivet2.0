# Play Audio Node

The Play Audio node sends an audio value to the active audio provider.

It is intended for interactive runs where Rivet has an audio playback provider available.

## Inputs

| Title | Data Type | Description           |
| ----- | --------- | --------------------- |
| Data  | `audio`   | Audio sample to play. |

## Outputs

| Title      | Data Type | Description                          |
| ---------- | --------- | ------------------------------------ |
| Audio Data | `audio`   | The same audio value after playback. |

## Settings

This node has no settings.
