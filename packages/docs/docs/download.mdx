# Download

## Latest Stable Release

Stable releases are built from the `main` branch and are the recommended downloads for usual desktop app users. Rivet publishes stable Windows installers and a macOS disk image here.

## Latest Developer Release

Developer releases are built from the `develop` branch. They are useful for trying the newest Rivet 2 work before it becomes the stable release, with Windows and macOS desktop builds published when the branch workflow succeeds.
